# redmine_codemirror

CodeMirror integration plugin for Redmine editor (wiki/issues).

Features:
- Syntax highlighting + CSS for headers
- Folding (Ctrl-Q or sidebar)
- Fullscreen (F10/esc)
- TODO marker, macro syntax
